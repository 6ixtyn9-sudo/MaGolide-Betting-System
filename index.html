Ma Golide v1.1.0

Advanced Sports Betting Prediction & Analysis System — Google Apps Script

How to Use This System

This project runs entirely inside Google Sheets using Google Apps Script. To deploy it:

  • Open your Google Sheet and navigate to Extensions → Apps Script.
  • Copy each .gs file from the docs/ folder into the Apps Script editor.
  • Run Sheet_Setup first to initialize the spreadsheet structure.
  • Use the custom menus generated by each module to interact with the system.

Alternatively, use Clasp to push files directly: clasp push

System Modules

Module 1

Sheet Setup

One-click creation of the full spreadsheet infrastructure. The "Single Source of Truth" for all sheet names and layout. Run this first.

Module 2

Config & Ledger Satellite

Manages system configuration, sport-specific settings (NBA, NFL, Volleyball), and betting ledger management.

Module 3

Signal Processor

The "Mouth" of the system. Cleans and standardizes messy raw sports data into a structured horizontal format ready for analysis.

Module 4

Data Parser

Specialized parsing engine using advanced regex and DFS-based splitters to decode concatenated team/date strings like DallasCowboysPhiladelphiaEagles.

Module 5

Margin Analyzer

Point spread and margin prediction engine. Analyzes historical margins to forecast home/away advantage and betting edges.

Module 6

Forecaster

Trend analysis and initial forecasting layer. Provides probability models feeding into over/under and quarter predictions.

Module 7

Game Processor

Production-ready prediction logic for Over/Under and Quarter outcomes. Features the Tuner Brier accuracy calibration system.

Module 8

Inventory Manager

Tracks the full inventory of games and betting opportunities across sports. Manages game states from upcoming to settled.

Module 9

Accumulator Builder

Assembles multiple picks into parlays/accumulators. Includes ROBBERS upset detection, confidence tier system (Elite/Strong/Medium/Weak), and first-half 1x2 predictions.

Audit

Contract Enforcer

Data integrity enforcement engine. Ensures all modules follow system rules, validates outputs, and flags violations automatically.

Audit

Contract Enforcement

Full audit trail from predictions → bet slips → results → accuracy reports. Calculates hit rates and tracks betting performance.

Key Features

Resilient Parsing

Handles "unkillable" data formats where teams and dates are concatenated without spaces or delimiters.

Multi-Sport Support

Supports NBA, NFL, and Volleyball with configurable scoring modes (points vs. sets).

Advanced Analytics

Implements shrinkage, continuity correction for integer lines, and conditional win rate calculations.

Automated Audit

End-to-end tracking from predictions through to settled results with hit rate reporting.
